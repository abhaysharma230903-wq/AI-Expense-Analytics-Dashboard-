# AI Expense Analytics Dashboard

AI Expense Analytics Dashboard is a machine learning-based finance management application designed to help users track expenses, analyze spending patterns, and predict future expenses.

The application combines expense tracking features with AI-powered analytics to provide insights into user spending behavior. It supports expense categorization, monthly reports, dashboards, and predictive analysis for better financial planning.

## Features

- User expense management and tracking
- Automated expense categorization using Machine Learning
- Monthly and category-wise spending analysis
- Expense trend visualization and reports
- Future expense prediction
- Interactive dashboard and charts
- SQL database integration
- Responsive web interface
